ESP32-S3 低功耗手持智能终端



独立完成基于 ESP32-S3 的嵌入式系统开发，涵盖驱动层、任务调度、低功耗管理与 OTA 可靠性设计。

🔹 驱动与中断设计

移植 ILI9341（SPI + DMA）与 XPT2046 触摸控制器
ISR 仅做事件通知与限频门控，SPI 坐标读取与去抖在任务上下文完成，避免中断长时间占用与总线互斥冲突
LEDC PWM 控制背光亮度，结合系统空闲状态动态调节功耗

🔹 GUI 与手势状态机

基于 LVGL 实现多页面 UI（桌面 / 设置 / 锁屏）
自研手势识别状态机（Tap / Double-tap / Long-press / Swipe）
时间与位移双阈值判定
双击延迟确认机制避免与单击冲突
长按与滑动优先级控制避免误判
动态刷新策略（1~33Hz），降低空闲 CPU 占用约 70%

🔹 线程池与任务调度优化

设计固定 worker 数线程池复用栈资源，限制最大并发任务数量
对可能阻塞的任务设置超时与隔离策略，避免单任务阻塞导致系统抖动
队列背压与任务合并机制防止任务积压

🔹 低功耗管理

构建发布-订阅式电源事件模型，外设自动响应 Light / Deep-sleep
进入 Deep-sleep 前冻结任务提交并按 deadline 排空关键任务，丢弃非关键任务，保证功耗目标与系统一致性

🔹 OTA 可靠性设计

HTTPS 双分区滚动升级
启动自检（Wi-Fi 可达性 + 堆健康检测）
失败自动回滚，避免固件异常导致设备不可用
Wi-Fi 指数退避重连策略，降低异常场景功耗
